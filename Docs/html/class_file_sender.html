void FileSender::send ( )
slot

Send file on specified host and port.

At this slots open file for reading transfering information, open Tcp socket and try to connect at specified host and port. After we initialize block and while transfer data more than block part of file size of block will send.

Warning
Sending may be failed if transfering file not open or socket can't connect to host. Whatever in this case finished() signal will be emited.
