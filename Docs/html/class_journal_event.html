Detailed Description

Journal event class.

Represent class for storage information about events fields like:

  • Type of Journal
  • Source host
  • Source port
  • Event ID
  • Machine Name
  • Data
  • Index
  • Category
  • Category Number
  • Entry Type
  • Message
  • Source
  • InstanceId
  • Time Generated
  • Time Written
